# Custom Fields User Guide

## How to Use Multiple Custom Fields in Classic Resume

### Adding Your First Custom Field

1. **Open Classic Resume**

   - From the home screen, tap "Classic Resume"
   - Scroll to the bottom of the form

2. **Find Custom Fields Section**

   - Look for the "Custom Fields" section
   - You'll see an "Add Custom Field" button

3. **Create a Custom Field**

   - Tap "Add Custom Field"
   - A new field card will appear with two inputs:
     - **Field Label**: Name your section (e.g., "Volunteer Work", "Publications", "Awards")
     - **Content**: Enter the details (multi-line supported)

4. **Save Your Work**
   - Changes save automatically as you type
   - No need to manually save!

## Adding Multiple Custom Fields

### Step-by-Step

1. Click "Add Custom Field" - First field appears
2. Fill in Label: "Volunteer Work"
3. Fill in Content: Your volunteer experience
4. Click "Add Custom Field" again - Second field appears
5. Fill in Label: "Publications"
6. Fill in Content: Your publications list
7. Repeat as many times as needed!

### Tips for Multiple Fields

- ✅ You can add unlimited custom fields
- ✅ Each field is independent
- ✅ Delete any field without affecting others
- ✅ Fields appear in order added (top to bottom)

## Editing Custom Fields

### Change Content

- Simply click into the text field
- Edit the label or content
- Changes save automatically

### Delete a Field

- Find the field you want to remove
- Click the **red trash icon** in the top-right
- Field is deleted immediately

## Where Custom Fields Appear

### Preview

- Open "Preview" from the form
- Scroll to the bottom
- Custom fields appear **after Certifications**
- Each field shows as a separate section with your label as the header

### PDF Export

- Download PDF from Preview
- Custom fields included at the end
- Professional formatting matching rest of resume
- Prints perfectly on A4 paper

### DOCX Export

- Download DOCX (Word) from Preview
- Custom fields appear as text sections
- Edit further in Microsoft Word if needed

### TXT Export

- Download TXT for ATS-friendly version
- Custom fields appear as plain text sections
- Great for copy-pasting into online applications

## Best Practices

### DO ✅

- Use clear, professional labels
- Keep content concise and relevant
- Use bullet points for lists
- Proofread before exporting
- Add only relevant custom sections

### DON'T ❌

- Leave label or content empty (will be skipped in exports)
- Use vague labels like "Other" or "Miscellaneous"
- Duplicate sections already in the form (e.g., don't add "Experience" as custom field)
- Add too many fields (5-7 is usually sufficient)

## Troubleshooting

### Custom Field Not Appearing in Preview?

- Check that both label AND content are filled in
- Empty fields are automatically skipped
- Tap "Save" if you added the field just now
- Close and reopen Preview to refresh

### Custom Field Not in PDF?

- Verify field appears in Preview first
- Empty label or content will be skipped in exports
- Try downloading again
- Check you're viewing the latest PDF (not an old cached version)

### Lost Custom Fields After Closing App?

- Custom fields save automatically as you type
- If you force-closed the app while typing, last few characters might be lost
- Always wait a second after typing before closing
- For cloud users: Check internet connection for auto-sync

### Too Many Fields - Preview Looks Crowded?

- Consider combining related fields
- Use bullet points instead of multiple fields for short items
- Remember: Hiring managers prefer concise resumes
- Aim for 5-7 custom sections maximum

## FAQ

**Q: How many custom fields can I add?**  
A: Unlimited! However, we recommend 5-7 for readability.

**Q: Can I reorder custom fields?**  
A: Not yet - they appear in the order added. Delete and re-add to change order.

**Q: Will custom fields work with other resume templates?**  
A: Currently only Classic template supports custom fields. Other templates coming soon!

**Q: Do custom fields sync to cloud?**  
A: Yes! If you're logged in, custom fields sync with all your resume data.

**Q: Can I use formatting (bold, italic) in custom fields?**  
A: Not yet - plain text only. Formatting features coming in future updates.

**Q: What if my custom field label is very long?**  
A: Labels up to ~50 characters work fine. Longer labels may wrap in PDFs.

**Q: Can I add images or links in custom fields?**  
A: Not currently supported - text only for now.
