User Guide

Quick setup for core Resumix AI features.

Create a Resume

  1. Open the app and create a new resume from the home screen.
  2. Fill in personal details, summary, experience, education, skills, and projects.
  3. Pick a template and preview the result before exporting.

Export as PDF

  1. Open the resume you want to export.
  2. Use the export or share action in the app bar.
  3. Choose PDF to generate a shareable resume file.

Use AI Features

  • AI tools are optional and only run when you trigger them.
  • Review all AI-generated content before using it in a real application.

Backup and Restore

  • Backup & Sync is optional.
  • Your resume data stays local unless you explicitly back it up.
