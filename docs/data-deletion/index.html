Data Deletion Instructions

Last updated: 2026-05-13

Resumix AI lets users delete their app data at any time. This page explains how to delete local data stored on the device and how to request deletion of synced backups that were created through optional backup or sync features.

Delete Data Inside the App

  1. Open Resumix AI.
  2. Go to Settings or Privacy & Security.
  3. Tap Delete My Data or Delete All Data.
  4. Confirm the deletion request when prompted.

This removes local app data from the current device and signs the user out. If the user has optional synced backups associated with the current device or sync code, those backups are also included in the deletion flow described in the app.

Delete Data by Email Request

If you cannot access the app and want assistance with a deletion request, email seenaigmk@gmail.com from the address associated with your account or include enough detail for us to identify the request safely.

  • Subject suggestion: Resumix AI Data Deletion Request
  • Include the sign-in method you used, such as phone OTP, Google, or Facebook
  • Include any relevant sync code or account email if available

Response Timing

We review deletion requests as quickly as reasonably possible after verifying the request. If more information is needed to locate the correct data, we may reply to confirm identity before completing the deletion.
